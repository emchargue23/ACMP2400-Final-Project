This project builds, scans, and deploys a containerized application to Azure using Terraform and GitHub Actions, creating a fully automated end-to-end CI/CD pipeline.

The project is run through a workflow triggered by workflow_dispatch. To run it, go to Actions -> Build-Curl-Test-Deploy-Test-Undeploy -> Run workflow.

AI DISCLOSURE:

I used AI in this project primarily for debugging and occasionally re-writing lines of code that needed to be fixed. I used Claude AI to assist me with most of the troubleshooting.
